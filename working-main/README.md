# GoldenSage
Goldensage is designed as a comprehensive digital companion, meticulously crafted to cater to the diverse needs of senior citizens. Our approach integrates technology with compassionate care, providing a seamless and empowering experience.

## Getting Started

### Prerequisites

- Python 3.8+
- MongoDB installed and running locally

### Installation

1. Create a virtual environment (optional but recommended):
   ```bash
   python -m venv venv
   # On Windows:
   venv\Scripts\activate
   # On macOS/Linux:
   source venv/bin/activate
   ```

2. Install dependencies:
   ```bash
   pip install -r requirements.txt
   ```

3. Run the application:
   ```bash
   python app.py
   ```

4. Open the application in your browser at `http://127.0.0.1:5000`.
